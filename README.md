# VEAI-OULAD-Q1

## Variational Educational Artificial Intelligence for Trustworthy and Equilibrium-Aware Learning Analytics

This repository provides the official implementation of the VEAI framework for educational intelligence using the Open University Learning Analytics Dataset (OULAD). The framework integrates variational-equilibrium modeling, educational stability analysis, learner-state dynamics, robustness evaluation, calibration analysis, and adaptive educational governance.

## Core Contributions

- Variational Educational Artificial Intelligence (VEAI)
- Equilibrium-aware educational intelligence
- Temporal educational dynamics and learner-state evolution
- Early-warning educational instability analysis
- Variational energy landscape modeling
- Robustness and calibration analysis
- Conventional AI versus VEAI comparative evaluation
- Adaptive educational governance and intervention mapping

## Dataset

The experiments use the Open University Learning Analytics Dataset (OULAD).

## Scientific Perspective

Unlike conventional educational AI systems focused solely on predictive optimization, VEAI models educational systems as dynamic equilibrium processes governed by competing educational forces including engagement, persistence, overload, disengagement, and educational stability.
